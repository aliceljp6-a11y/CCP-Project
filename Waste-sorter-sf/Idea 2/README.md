# Trash Simulation!

A small educational web app for 4th graders: see how 24 hours of San Francisco’s waste fills a transfer station. Built with React, Matter.js, and Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (e.g. http://localhost:5173).

## Deploy to Vercel (via GitHub)

See **[DEPLOY.md](./DEPLOY.md)** for step-by-step instructions to push this project to GitHub and deploy it on Vercel.

## Features

- **Transfer station** – Gray pit view where trash falls and stacks (Matter.js).
- **24-hour clock** – Top-right; 1 hour = 5 seconds real time.
- **Trash rain** – Blue (recycling), green (compost), and black (landfill) shapes fall and pile up.
- **Scale** – A “4th Grader” silhouette (~4.5 ft) at the bottom for size reference.
- **Screen shake** – When the pile reaches knees, waist, or head.
- **Start** and **Reset** – Control the simulation.
- **End card** – After 4:00 PM, a popup with facts and a link to Recology to learn more.

## Stack

- React 18
- Matter.js (2D physics)
- Tailwind CSS
- Vite
